MongoDB基本语法

发表于 2019-04-10 | 分类于 DataBase ， MongoDB

SQL术语 VS MongoDB术语

SQL术语 MongoDB术语 解释
database database 数据库
table collection 数据库表/集合
row document 数据记录行/文档
column field 数据字段/域
index index 索引
primary key primary key 主键，在MongoDB中自动将_id设为主键

基本语法

语法 解释
show dbs 查看数据库
show collections 查看集合
- -
use demo 创建数据库
db.createCollection(“user”) 创建集合
db.users.insert({id:123,name:”zl”}) 创建集合并插入数据
- -
db.dropDatabase() 删除数据库
db.user.drop() 删除user集合
db.user.remove({userId:101}) 删除文档
- -
db.user.find() 显示集合中的数据
db.user.find().pretty() 格式化显示集合中的数据
db.user.findOne() 查看第一条数据
db.user.update({userName:”Jack”},{$set: {userAge:30}}) 更新Jack的年龄
db.user.find({userName:”Jack”}) 查找Jack
db.user.find({userAge:{$gt:20}}) 查找年龄大于20 >
db.user.find({userAge:{$gte:20}}) 查找年龄大于20 >=
db.user.find({userAge:{$lt:20}}) 查找年龄小于20 <
db.user.find({userAge:{$lte:20}}) 查找年龄小于等于20 <=
db.user.find({userAge:{$eq:20}}) 查找年龄大于等于 =
- -
mongoimport -d filename -C collectionname –file filepath 导入
