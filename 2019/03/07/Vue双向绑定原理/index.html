Vue双向绑定原理

发表于 2019-03-07 | 分类于 Vue

原理

VUE实现双向数据绑定的原理利用了Object.defineProperty()方法,重新定义了对象获取属性值get和设置属性值set的操作来实现的。

语法

  • obj: 要定义的对象
  • prop: 要定义或修改的属性的名称
  • descriptor：将被定义或修改的属性的描述符
